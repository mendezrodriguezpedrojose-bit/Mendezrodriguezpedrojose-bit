# 💫 About Me:
Ingeniero Electrónico en Computación apasionado por la convergencia entre el hardware y el desarrollo de software. Mi enfoque principal está en el desarrollo de aplicaciones de escritorio y herramientas de optimización utilizando Python, diseñando arquitecturas limpias y experiencias de usuario fluidas mediante frameworks modernos. Siempre buscando optimizar el rendimiento, automatizar procesos complejos y construir código robusto y mantenible.
